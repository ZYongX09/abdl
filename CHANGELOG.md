# ABDL Space 更新日志

## v5.1.6 — 2026-05-09

### ✨ 新功能
- **图片灯箱**：帖子图片、评论图片、子评论图片均可点击放大查看，支持 Escape 键关闭
- **灯箱动画**：半透明模糊遮罩层 + 居中缩放动画，右上角圆形关闭按钮 hover 放大

### 🎨 UI/UX 优化
- **交错入场动画**：论坛帖子列表、首页纸尿裤卡片、排行榜列表加载时逐个滑入（stagger fadeInUp）
- **个人中心选项卡指示器**：Tab 下划线平滑跟随当前选中项移动，动画过渡流畅
- **深色模式 placeholder**：输入框占位符颜色加深，提升深色主题下的可读性

### 📝 重构
- `Profile.jsx` — 选项卡重构为 `PROFILE_TABS` 配置数组 + ref-based 指示器定位
- `PostDetail.jsx` — 新增 lightbox 状态管理 + Escape 键监听
- `App.css` — 新增 `.stagger-item`、`.lightbox-overlay`、`.lightbox-close` 样式

## v5.1.5 — 2026-05-09

### ✨ 新功能
- **用户主页**：新增 `/user/:id` 个人主页，展示头像、用户名、角色标签、加入日期
- **用户统计**：主页展示帖子数、收到的评论数、获得的点赞数
- **用户帖子列表**：主页可查看该用户发布的所有帖子，点击跳转帖子详情
- **评论图片上传**：帖子评论支持上传图片，选择后显示缩略图预览，可点击移除
- **评论字符计数**：评论框实时显示字符数（上限 500），接近上限时颜色警告

### 🎨 UI/UX 优化
- **用户主页骨架屏**：加载用户信息时显示骨架屏动画
- **用户主页选项卡**：帖子 Tab 带下划线指示器，空状态时显示友好提示
- **资料项图标化**：年龄/地区/体重等每项资料前添加 Font Awesome 图标
- **评论图片上传按钮**：使用主题按钮样式，替代原生文件选择器
- **评论 Markdown 提示**：评论框下方显示「支持 Markdown」提示文字
- **评论图片展示**：评论和子评论中的图片自动渲染展示（max 200px）

### 📝 文件变更
- `UserPage.jsx` — 新建用户主页组件（骨架屏 + 统计 + 帖子列表 + 选项卡）
- `PostDetail.jsx` — 评论图片上传预览 + 字符计数 + Markdown 提示 + 图片渲染
- `App.jsx` — 新增 `/user/:id` 路由 + 页面标题映射
- `api.js` — forumAPI.comment 数据模型新增 `image_url` 字段
